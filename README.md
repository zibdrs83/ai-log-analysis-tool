# 🔐 AI Log Analysis Tool

This project analyzes system logs to detect suspicious login activity and generate security insights.

## 🚀 Features
- Detects failed, invalid, and successful login attempts
- Tracks IP-based activity
- Identifies suspicious patterns
- Generates structured security reports
- Integrates offline AI for automated analysis

## 🛠️ Tech Stack
- Python
- Log Analysis
- Basic SIEM Concepts

## 📂 How It Works
The script reads a log file, parses login attempts, tracks activity per IP address, and flags suspicious behavior based on patterns.

## 🎯 Goal
To simulate real-world security monitoring and incident detection using automation.

- Add a dashboard (UI)
- Integrate real-time log monitoring
